AI Taskforce — AI-Powered Workflow Automation Initiative
  • Context:

    An internal innovation project at Therap Services to streamline content creation workflows using Large Language Models (LLMs) and generative AI tools.

  • Problem:

    Creating Request for Proposals (RFPs), HTML-based release notes, user guides, and tutorial videos manually was time-consuming and error-prone. We needed automated solutions that could interpret diverse content formats (text, documents, screenshots) and generate consistent, accurate outputs with minimal human intervention.

  • My Contribution:

    • Designing prompt-driven workflows using Gemini Gems and ChatGPT to automatically draft RFPs from structured customer requirements.
    • Developing an AI-assisted system to convert release notes and user guide documents (with screenshots) into clean HTML for our support site.
    • Creating AI-based video generation workflows using Heygen and NotebookLM to produce “Therap Shorts” tutorial videos.
    • Engineering and optimizing prompts across multiple AI tools to achieve consistent, high-quality outputs.

  • Technologies Used:

    Gemini, ChatGPT, NotebookLM, Heygen, Prompt Engineering

FocusGRE (In Progress, Personal Project)
  • Context:

    I started this project to improve my GRE preparation experience and to practice full-stack web development.

  • Problem:

    Most existing GRE apps are either costly or lack adaptive, engaging features that support dynamic learning.

  • Solution (Vision):

    • I am building a full-stack web app that provides:
      • Quant & Verbal practice (topic-wise + random tests).
      • Interactive solutions with explanations.
      • Vocabulary learning with flashcards, spaced repetition, and difficulty tiers.

      • Guest & signed-in user modes (with saved progress)

  • My Contribution:

    • Built dynamic question handling with JSON.
    • Implementing performance analytics and progress tracking.

  • Technologies Used:

    HTML, CSS, JavaScript

    • This is an ongoing project. The core sections (Quant, Verbal & Mock Test) are functional, while Vocabulary, Practice Questions, and Profile are under development. Planned completion: September 2025.
